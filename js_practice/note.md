# 자바스크립트 학습

## 세팅

### 웹사이트(html)에 자바스크립트 넣는 법

### 인라인 방식

```
<script>
    alert('Hello)
</script>
```


### js 파일 방식

```
<script src="script.js></script>
```

### script 태그의 defer 속성

이 속성은 html이 모두 로드된 이후 js가 실행하도록 설정한다.

### type 속성

`type="module"`을 이용하면 자바스크립트 모듈로 취급된다.

이를 이요하면 import와 export를 사용할 수 있다.

#### export 방식

`export 변수/함수 선언;`

`export default "asdf"` => 해당 파일 기본 export 값 설정. default 출력값은 파일당 1개만 존재 가능


```
export let apiKey = "asdf";

export default "asdf";

```

#### import 방식

`import { 변수명, 변수명1 } from 파일 위치;`

`import 변수명 frrom 파일 위치;` => 디폴트 export 값을 받을 때에도 변수명 지정 필요.

`import * as 받을 객체명 from 파일 위치;` => 객체로 import 하기

```
import { apiKey, abc as content } from "./util.js"; // 리액트에서 사용할 떄는 js 학장자 빼도 됨 (자동추가)

import 변수명 frrom 파일 위치;

import * as 받을 객체명 from 파일 위치;
```

### Build Process를 사용하는 이유

`package.json`: 프로젝트에서 사용되는 의존성이 선언되어 있음.

1. 의존성 라이브러리 연결
2. 코드의 난독화 및 간략화

## 변수 및 값

### 변수 타입

1. 문자형 (String)
2. 숫자형 (Number)
3. 논리형 (Boolean)
4. 미정 (Null & undefined)

### 선언 방식

let 변수명
const 변수명

## 연산자

`==` => 타입이 다를 경우 자동 형변환 이후 비교
`===` => 형변환 없이 피연산자가 정확히 일치해야 true 반환

## 함수 매개변수

### 선언 방법

`function 함수명(매개변수)`

### 사용 방법

`함수명(인자)`

## 화살표 함수

익명 함수를 활용하면, 짧고 간결하게 함수를 선언할 수 있습니다.

`(변수명) => { 함수 내용 }`

오직 하나의 매개변수만을 받을 경우 묶는 괄호를 생략할 수 있습니다.

`변수명 => {}`

단, 매개변수가 없는 경우, 괄호를 생략할 수 없습니다.

`() => {}`

### 중괄호 생략

함수 본문에 반환문 외 다른 로직이 없을 경우, return 키워드와 중괄호를 생략할 수 있습니다.

`(변수명) => 반환값`

## 객체

중괄호를 이용하여 객체를 생성할 수 있습니다.

```
const aClass = {
    name = "철수",
    age = 345,
    greet() {
        console.log("Hello");
        console.log(this.age);
    }
}
```

또한 class 키워드를 이용하여 클래스를 미리 정의해놓을 수 있습니다.

`class` 키워드를 사용한 이후 클래스의 이름을 지정합니다.

`class ClassName`

이후 중괄호를 이용하여 클래스 영역을 선언한 뒤, 해당 영역에 내용을 표기합니다.

`constructor` 키워드를 사용하여 해당 객체가 가질 데이터를 정의하고 초기화 합니다.

`constructor(변수명) => this.변수명 = 변수명`

```
class User {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log("Hello");
        console.log(this.age);
    }
}
```

## 배열

여러개의 데이터를 표현하기 위해 배열을 사용할 수 있습니다.

배열은 대괄호를 이용하여 표현합니다.

배열 내부의 데이터를 이용할 땐 `변수명[변수 위치]`와 같이 사용합니다.

```
const hobies = ["Sports", "Cooking"];
console.log(hobies[0]);
```

배열에는 몇 가지 편리한 함수가 내장되어 있습니다.

`push` : 마지막에 원소를 추가

`findIndex` : 조건에 맞는 첫 원소의 인덱스를 반환

`map` : 배열을 순회하며 새로운 배열을 생성해 반환


## 배열과 객체의 분해

모던 자바스크립트 문법 중 객체와 배열의 원소를 간편하게 사용할 수 있는 분해라는 것이 있습니다.

아래 코드를 보면, 배열의 첫 번째 원소와 두 번쨰 원소를 다른 이름의 변수로 저장하고 있습니다.

```
const array = [1, 2];

const first = array[0];
const second = array[1];

const person = {
    name = "철수",
    age = 10
}
const name = person.name;
const age = person.name;
```

해당 코드를 분해를 이용한다면 더 간편히 표현할 수 있습니다.

```
const [first, second] = [1, 2];
const {name, age} = {
    name = "철수",
    age = 10
}
```

단, 이 때 주의할 점은 객체를 분해할 경우, 기본적으로 분해해 얻을 변수의 이름을 객체의 key와 동일하게 맞추어야 합니다.

만약, 다른 이름을 지정하고 싶다면, `:`을 이용하여 새로운 이름을 지정할 수 있습니다.

아래 코드는 name 대신 userName으로 변수명을 지정한 예시입니다.

```
const {name: userName, age} = {
    name = "철수",
    age = 10
}
```

이를 응용한다면 함수의 매개변수를 받을 때, 편리하게 사용할 수 있습니다. 

아래 코드의 경우, 객체를 매개변수로 전달받지만, 분해를 이용하여 더 편리하게 매개변수를 활용합니다.

```
// 함수 호출문
storeOrder({id: 5, currency: 'USD', amount: 15.99}); // 1개의 매개변수 / 

// 함수 정의문
// 분해를 이용하여 객체.필드명 대신 간편하게 데이터에 접근할 수 있다.
function storeOrder({id, currency}) { // 디스트럭처링
  localStorage.setItem('id', id);
  localStorage.setItem('currency', currency);
}
```

## 배열의 전개

배열의 경우 데이터의 묶음이라고 생각할 수 있습니다.

자바스크립트의 `...` 연산자를 이용하면 필요한 경우, 배열의 데이터를 풀어서 사용할 수 있습니다.

예를들어 지갑에 존재하던 돈을 새로운 지갑으로 옮긴다고 생각해봅시다.

새로운 동전 지갑에 기존 동전 지갑의 내용물을 쏟아버릴 수 있을 겁니다. 이와 유사하게 동작합니다.

아래 코드의 경우, 새로운 지갑엔 새로운 지갑에 들어있던 돈(1000)과 더불어, 기존 지갑에 돈이 포함되는 것을 볼 수 있습니다.

```
wallet = [100, 50, 500];
myNewWallet = [1000, ...wallet];

console.log(myNewWallet);
// 출력값: [1000, 100, 50, 500]
```

이를 객체에도 동일하게 사용할 수 있습니다.

```
const user = {
    name = "철수",
    age = 10
};

const extendUser = {
    isAdmin = true,
    ...user
};
```


## 제어문

제어문은 조건에 만족할 경우 특정 기능을 실행하도록 합니다.

`if`, `else`, `else if`가 존재합니다.

`if`는 조건에 맞을 경우 다음 구문을 실행합니다.

`else`는 if와 함꼐 쓰여야하며, if 조건이 불만족 할 떄 실행됩니다.

`else if`는 if 조건과 함께 쓰여야하며, if 조건이 불만족 하고 새로운 조건을 만족한 경우 구문을 실행합니다.


## 반목문

반복문은 배열과 같은 여러 데이터가 묶여있는 구조를 이용할 떄 편리하게 사용할 수 있습니다.

아래와 같은 사용법들이 존재합니다.

1. 일반 for (var i = 0; i < 10; i++)
2. while
3. for (속성명 in 객체) => 속성명 순회
4. for (요소 of 컬렉션) => 컬렉션 순회
5. forEach


## 함수를 변수로 사용하는 방법

자바스크립트는 함수형 언어를 지원하여, 함수를 변수로 넘기고 받을 수 있다.

매개변수로 넘길 경우 괄호를 붇히지 않는다.

매개변수로 전달 받을 경우는, 일반 매개변수와 같이 전달받을 수 있으며, 해당 매개변수가 함수라면, 일반 함수를 사용하는 것처럼 `변수명()`으로 사용할 수 있다.

```
const myHandler = () => 1;

setTimeOut(myHandler, 200);
```


## 함수 내부에 함수 선언

자바스크립트에서는 함수 내부에 함수를 선언할 수 있습니다.

내부 선언된 함수에서는 해당 함수가 선언된 부모 함수의 변수들을 이용할 수 있습니다.


## 참조형 값과 기본 값

기본형 값은 변수 그 자체가 저장되지만, 참조형 값은 메모리 주소가 저장된다고 볼 수 있다. (정확한 설명은 아님)

따라서 참조형 값은 기본적으로 해당 주소 메모리 값의 데이터가 변경되더라도 참조하는 데이터 주소가 일치하더라면 값이 변경되었다고 보지 않는다.
